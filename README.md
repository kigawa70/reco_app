## アルゴリズム
- 選択あり：アイテムベース協調フィルタリング（コサイン類似度）。
  - 各選択映画の類似度ベクトルを合算 → 総合スコア → 選択済み除外 → 上位5件を返す。
- 選択なし：全ユーザの平均評価が高い映画上位5件を返す。

## ロジック（ボタン押下から表示まで）
1. フロントで選択された movieId 配列を `/api/recommend` に POST。
2. バックエンドは selected が空か否かで分岐：
   - 非空 → 合算したアイテム類似スコアで推薦
   - 空 → 平均評価ランキングで推薦
3. JSON でタイトルとスコアを返し、フロントで表示。

## 起動
1. pip install flask pandas numpy
2. python app.py
3. ブラウザで http://localhost:5000
