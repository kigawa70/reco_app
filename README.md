映画レコメンデーションシステム
概要

本システムは、ユーザが選択した映画の傾向に基づいて
おすすめ映画を動的に表示する Web アプリケーションである。
映画データおよび評価データを用いてレコメンデーションを行う。

使用技術
Python
Flask
pandas
NumPy
HTML

レコメンデーションアルゴリズム
好きな映画を選択した場合
アイテムベース協調フィルタリング を採用した。
・映画 × ユーザの評価行列を作成
・映画同士のコサイン類似度を算出
・選択された3つの映画それぞれの類似度スコアを合算
・選択済み映画を除外し、上位5件を推薦

好きな映画が未選択の場合
・評価平均ベースのランキング方式 を採用した。
・全ユーザの評価データから映画ごとの平均評価を算出
・評価が高い映画を上位5件表示

処理の流れ
ユーザが映画を最大3つ選択
「オススメ映画を表示」ボタンを押下
バックエンドで選択有無に応じて処理を分岐
おすすめ映画5件を取得し画面に表示

起動手順
pip install flask pandas numpy
python app.py


ブラウザで以下にアクセスする。

http://127.0.0.1:5000

工夫した点

複数の映画を選択した場合、それぞれの類似度を合算することで
ユーザの嗜好を総合的に反映できるようにした

映画未選択時でも必ずおすすめが表示されるよう処理を分岐した
